# Piece set attribution

**Set:** Fantasy (Staunton-style)
**Author:** [Maurizio Monge](https://github.com/maurimo/chess-art)
**License:** MIT

The 12 SVG piece files (`wK.svg`, `wQ.svg`, `wR.svg`, `wB.svg`, `wN.svg`,
`wP.svg`, `bK.svg`, `bQ.svg`, `bR.svg`, `bB.svg`, `bN.svg`, `bP.svg`)
in this directory come from the author's `chess-art` repository and
are redistributed here under the MIT license (see the upstream
[LICENSE](https://github.com/maurimo/chess-art/blob/main/LICENSE)).

We selected this set because it is one of the few chess piece sets
with an MIT-compatible license — most popular sets (Cburnett, Merida,
Horsey, etc.) are GPL-2+ or CC-BY-NC-SA which are incompatible with
this project's MIT license. Alternative MIT options from the same
author include `spatial` and `celtic`; the free CC0 option is
`rhosgfx` by RhosGFX.

The files were obtained via `curl` from the mirrored copies hosted in
the [lichess-org/lila](https://github.com/lichess-org/lila/tree/master/public/piece/fantasy)
repository and are unmodified.
